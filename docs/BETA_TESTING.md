# HABoard Beta Testing Program

We're looking for 5-10 beta testers to help make HABoard even better!

## What is HABoard?

HABoard is an offline-first To-Do PWA for Home Assistant that lets you manage tasks with:
- ‚úÖ Full offline support
- ‚úÖ Real-time sync across devices
- ‚úÖ Beautiful PWA interface
- ‚úÖ Keyboard shortcuts (11!)

- ‚úÖ WCAG 2.1 accessible

## What We're Looking For

### Requirements

**Must Have**:
- Home Assistant 2024.1.0 or newer

## Testing Scenarios

### Scenario 1: Daily Task Management
1. Create 3 tasks for today
2. Set priorities and due dates
3. Complete 1 task
4. Edit 1 task
5. Delete 1 task
6. Verify sync across devices

1. Disconnect from internet
2. Create 2 new tasks

5. Reconnect to internet
6. Verify all changes synced

### Scenario 3: Tag Organization
1. Create 3 tags with different colors

3. Search by tag
4. Delete a tag
5. Verify tag removed from tasks

### Scenario 4: Mobile Experience
1. Install PWA on phone
2. Use swipe gestures

4. Try keyboard shortcuts
5. Test offline on mobile

### Scenario 5: Multi-Device Sync
1. Create task on Device A
2. Verify appears on Device B
3. Edit on Device B
4. Verify changes on Device A
5. Test conflict resolution

## FAQ

### Can I test on multiple devices?
Yes! We encourage it. Multi-device testing is valuable.

### What if I find a critical bug?
Report immediately via GitHub Issues with "CRITICAL" in title.

### Can I invite others?
Not during closed beta. Public beta may come later.

### Will my data be safe?
Beta software can have bugs. Don't use for critical tasks. Regular backups recommended.

### Can I stop testing anytime?
Yes, absolutely. Just let us know.

### What happens to my test data?
It stays on your Home Assistant. You can delete it anytime.

### Will beta testers get early access to future features?
Yes! Beta testers get priority for future testing.
