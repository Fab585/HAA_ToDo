## Known Limitations (Intentional MVP Scope)

The following features are intentionally deferred to Beta (v0.5) or v1.0:

### Deferred to Beta (v0.5)
- No batch operations (select multiple tasks)
- No push notifications (Service Worker ready but not implemented)
- No haptic feedback on mobile
- No pull-to-refresh gesture
- No Home Assistant authentication (assumes trusted local network)
- No performance optimizations for 1000+ tasks

### Deferred to v1.0
- No natural language parsing (structured forms only)
- No calendar view (list view only)
- No recurring tasks
- No subtasks or dependencies
- No import/export functionality
